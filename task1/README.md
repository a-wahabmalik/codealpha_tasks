# CodeAlpha Task 1 - Network Packet Sniffer 
 
## Description 
A lightweight Python-based network traffic sniffer built using scapy to capture, inspect, and analyze packets in real time.
This project helps beginners understand how data flows through a network and how different protocols operate at the packet level.
